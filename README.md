# Aplikasi Peminjaman Barang Kominfostan.

## Deskripsi
Aplikasi berbasis PHP dan MySQL untuk mengelola peminjaman dan pengembalian barang di lingkungan Kominfostan.

## Fitur
- Login & Register
- Multi role (Admin & User)
- Peminjaman barang
- Pengembalian barang
- Manajemen data barang
- Validasi stok otomatis

## Teknologi
- PHP Native
- MySQL
- Bootstrap

## Cara Instalasi
1. Clone repository
2. Import file database.sql ke phpMyAdmin
3. Atur koneksi database di file database.php
4. Jalankan melalui XAMPP di localhost
